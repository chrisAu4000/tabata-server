# tabata-server

## installation
```bash
$ git clone https://github.com/chrisAu4000/tabata-server.git 
$ cd tabata-server
$ npm install
$ touch config.json
```

Add your sendsgrid API_KEY in config.json.
```json
{
  "SENDSGRID_API_KEY": "your api key."
}
```
## Requirement
mongodb, sendgrid API_KEY
